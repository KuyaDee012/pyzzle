    Introduction

    Pygame is a set of cross-platform Python modules designed for writing video games. It includes graphics and sound libraries designed for use with the Python programming language.

    Know more about Pygame →
    About PYZZLE

    Pyzzle is a free educational website for coding online. It focuses mainly on pygame module. After learning the recommended programming language, the user will now be able to create or develop a simple game according to his or her preferences.

    Game Examples

    Pacman

    Pac-Man is a maze chase action video game; The player controls the character of the same name through a closed maze. The goal of the game is to eat all the dots placed in the maze while avoiding four colored ghosts.

    Ball Game

    It is a very simple arcade style colliding ball game. The objective is to get as large a score as possible without the balls hitting the big ball at the bottom.

    Space Invaders

    It is a fixed shooter in which the player moves a laser cannon horizontally across the bottom of the screen and fires at aliens overhead.
